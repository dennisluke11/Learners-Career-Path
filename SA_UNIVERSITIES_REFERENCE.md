## Programs Offered by University Type

### Traditional Universities Typically Offer:
- Medicine (MBChB)
- Engineering (BEng)
- Law (LLB)
- Business (BCom)
- Science (BSc)
- Arts (BA)
- Education (BEd)

### Universities of Technology Offer:
- BTech degrees (4 years)
- National Diplomas (3 years)
- Higher Certificates (1-2 years)
- Focus on practical/vocational training

### Specialized Programs:
- **Veterinary Science**: Only at UP
- **Dentistry**: UCT, Wits, UP, UWC, SMU
- **Medicine**: UCT, Wits, UP, SU, UKZN, UFS, UL, SMU, NWU
- **Actuarial Science**: UCT, Wits, UP, SU, UKZN, UFS, NWU

## APS Score Guide

| APS Range | Competitiveness | Typical Universities |
|-----------|-----------------|----------------------|
| 40-42 | Very High | UCT, Wits (competitive programs) |
| 35-39 | High | UCT, Wits, UP, SU (most programs) |
| 30-34 | Medium-High | UP, SU, UKZN, UJ, NWU |
| 26-29 | Medium | UFS, NMU, RU, UWC |
| 22-25 | Medium-Low | UFH, UL, Univen, UniZulu, WSU |
| 18-21 | Open Access | UNISA, some UoT diplomas |

## Usage in App

Run the population script to add all universities:

```bash
# Preview changes
node scripts/populate-all-sa-universities.js --dry-run

# Apply changes
node scripts/populate-all-sa-universities.js
```

This will update all careers with comprehensive university data, including:
- All 26 public universities
- Accurate APS requirements per institution
- Regional options for students
- Both degree and diploma pathways
